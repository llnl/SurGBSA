# SurGBSA
Implementation of "SurGBSA: Learning Representations From Molecular Dynamics Simulations" available on [arXiv](https://arxiv.org/abs/2509.03084)


# Dataset, Model Checkpoints, Splits

Our dataset (MD trajectories, MMGBSA scores, and processed ML-ready files), model checkpoints, and splits are made available on huggingface: <https://huggingface.co/datasets/llnl/SurGBSA>
